問卷與參考文獻

問卷

基本資料

1.性別:

2.年齡:

3.起點:

4.終點:

5.使用目的:A 就學 B 就醫 C 觀光 D 購物 E USR F 其他

第一部分 改變(以下選項皆為李克特五點量表)

6.您是否瞭解本計畫推動的目的

7.使用共乘服務後，您在往返目的地的便利性上是否有具體改善

8.使用共乘後，您規劃外出的自主性提升

9.共乘有助於減輕交通負擔

10.使用共乘後您更常參與社區活動

11.共乘使我的生活變得更豐富

12.您感到更開心或滿足

第二部分 歸因

13.這次共乘服務確實解決了您的交通不便

14.如果您的生活交通獲得改善，您認為有多少比例能歸功在共乘服務上

第三部分 無異

15.如果沒有共乘服務，您是否還有意願參加本次活動

16.若有，您的替代交通工具為 (根據交通工具計算減碳量)

第四部份 替代

17.您是否認為本共乘排擠了其他交通工具營運

第五部分 量化

18.使用共乘後，本次外出節省時間

19.使用共乘後，本次節省交通費

20.這趟活動行程對您來說非常重要。若因為沒有這趟共乘車，導致您無法成行，請問您願意付出多少代價來換取這趟行程？(貨物替代社會價值)

第六部分 未來規劃

21.您願意支付多少金額享受共乘服務?

為避免受訪者產生回應順序效應，導致題目間產生虛假的關聯或誘導式回應，本研究參考 Krosnick & Alwin (1987) 關於認知處理與順序效應之理論，針對調查題項採取了順序隨機化設計，以確保每項指標在受訪者心中的評估皆具備獨立性，而非受到題序的認知引導。
